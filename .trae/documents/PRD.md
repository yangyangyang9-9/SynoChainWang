# SynoChain AI 产品需求文档 (PRD)

## 1. 产品概述

SynoChain AI 是全球首个社区共建的 AI 健康伴侣平台，通过 SEO 健康知识流量获取用户，AI 健康助手提供价值，最终转化为众筹会员和 App 用户，形成完整的商业闭环。
- **核心目标**: 构建 AI 健康助手 + 海量健康知识库的众筹平台，让个性化健康指导人人可及
- **目标用户**: 搜索健康信息的全球用户、关注自然健康方案的群体、愿意支持创新 AI 项目的早期支持者
- **市场价值**: 健康信息搜索是互联网最大需求之一，SEO + GEO（生成式搜索优化）是未来趋势

## 4. 用户界面设计

### 4.1 设计风格

- **主色调**: 纯白背景 (#FFFFFF) + 医疗蓝 (#0066FF) 作为主强调色 + 健康绿 (#00C896) 作为辅助强调色
- **次要色**: 深炭灰 (#1A1A2E) 文字 + 浅灰 (#F8F9FA) 背景分区
- **按钮风格**: 圆角按钮 (8px)，主按钮使用医疗蓝填充，次按钮使用描边
- **字体**: 显示字体使用 Sora（现代几何感），正文使用 Plus Jakarta Sans（清晰易读）
- **布局风格**: 桌面优先，顶部固定导航 + 卡片式内容网格，宽屏居中布局 (max-width: 1280px)
- **图标风格**: 线性图标 (Lucide style) + 适度 emoji 辅助情感表达
- **设计原则**: 模块化设计，每个功能区独立成模块，便于后期扩展

### 4.2 页面设计概览

| 页面名称 | 模块名称 | UI 元素 |
|----------|----------|----------|
| 首页 | Hero 区域 | 大标题渐变文字、副标题、CTA 按钮、众筹进度条、数字统计卡片 |
| 首页 | 资金用途 | 环形进度图 + 百分比卡片，四个用途分块展示 |
| 首页 | Founder 等级 | 双卡片对比布局，权益清单打勾，价格突出显示 |
| 首页 | 工具导航 | 3x3 网格卡片，每卡片含图标/标题/描述 |
| AI Health Tools | AI 对话 | 聊天气泡布局、输入框、建议问题芯片 |
| Disease Library | 列表/详情 | 字母索引 + 卡片列表，详情页分章节标签 |
| 通用 | 顶部导航 | Logo + 菜单 + 语言切换 + CTA 按钮 |
| 通用 | 页脚 | 链接列、社交媒体、版权信息、多语言切换 |

### 4.3 响应式设计

- 桌面优先设计（1280px+ 完整布局）
- 平板自适应（768-1280px 网格调整）
- 移动端适配（<768px 单列布局、汉堡菜单、触摸优化）
- 所有交互元素触摸友好（最小 44px 点击区域）

### 4.4 多语言切换功能

- 默认英语，支持中文、西班牙语、日语等
- 顶部导航语言切换器（下拉或图标）
- 所有 UI 文案、健康内容、AI 回复支持多语言
- 语言偏好持久化存储
